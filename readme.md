# cruftbusters/devpod

This is a super-repository for research, development and engineering at Cruftbusters. It includes a self-building devpod or devcontainer.

## prerequisites

1. git
2. podman

## start the devpod

Run exec with any command. For example:

1. ``` ./exec zsh ```
2. ``` ./exec nvim ```

## destroy the devpod

``` ./remove ```
